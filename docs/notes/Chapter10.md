1. 使用 proxyTable 配置项

```js
dev: {
  // Paths
  assetsSubDirectory: 'static',
  assetsPublicPath: '/',
  proxyTable: {
    // 当请求api目录的时候
    '/api': {
      // 把请求转发到服务器8080端口上
      target: 'http://localhost:8081',
      // 路径进行一个替换
      pathRewrite: {
        // 一旦请求的地址是以api开头的,就请求到本地的 static 目录下 mock 文件夹下
        '^/api': '/static/mock/'
      }
    }
  }
```
